Despre Noi
Despre Proiect
Aplicația iTEC Monitor a fost concepută atât pentru monitorizarea URL-ului dar și a Socket-urilor.
Despre Echipa
Echipa noastra este formata din 3 membri. Pe partea de backend cat si frontend se afla Mladin Alexandru, cu o experienta in java de peste 7 ani de zile. Adi este tot pe partea de frontend, cu o experienta de 5 ani de zile. Erik este incepator.
Detalii Tehnice
Backend-ul este facut in SpringBoot, cu un templating engine Thymeleaf, baza de date este Mysql, iar pe partea de frontend s-au folosit librariile javascript jquery si anim.
